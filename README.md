# IDEA
Geant4 simulation of the IDEA Preshower

*******************************

**IDEA** 
---
It contains the description of the geometry


 **IDEA-build** 
---
It contains the executable files.

-  To compile and execute:
   ```
   make
   ./IDEA -m IDEA.in
   ./IDEA -m vis.mac
   ```
-  To visualize using a grapich window:
   ```
   java -jar ../HepRApp.jar
   ```
